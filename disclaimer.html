Os resultados gerados por esta aplicação não substituem a responsabilidade técnica do profissional habilitado e não devem ser utilizados como documentação definitiva de projeto sem revisão e assinatura de engenheiro responsável registrado no CREA ou CAU.

1. Natureza e Finalidade da Ferramenta

O HidroFlow é uma aplicação web de código aberto destinada a auxiliar estudantes de engenharia, engenheiros e técnicos em saneamento no dimensionamento preliminar e estudo de concepção de Sistemas de Abastecimento de Água (SAA).

A ferramenta implementa metodologias consagradas e normas técnicas brasileiras como referência de cálculo. Contudo, toda aplicação técnica é inerentemente dependente de hipóteses de projeto, condições locais específicas e da expertise do profissional que a opera.

A utilização do HidroFlow pressupõe que o usuário possui conhecimento técnico suficiente para:

  • Interpretar os resultados criticamente à luz das condições locais do projeto;

4. Conformidade com Normas Técnicas

O HidroFlow implementa as seguintes normas como referência de cálculo. O usuário deve verificar se a versão da norma aplicável ao seu projeto é a mesma utilizada na ferramenta:

Norma Título Versão implementada
NBR 12.211 SAA — Estudo de Concepção 2017
NBR 12.217 Projeto de Reservatório de Distribuição 1994
NBR 12.218 Projeto de Rede de Distribuição de Água para Abastecimento Público 2017

As normas técnicas são revisadas periodicamente pela ABNT. O usuário é responsável por verificar a vigência das normas antes de utilizá-las em projetos oficiais.

5. Uso dos Dados da API do IBGE

Os dados populacionais carregados via API do IBGE/SIDRA correspondem aos Censos Demográficos e Contagens Populacionais oficiais. O HidroFlow não modifica nem interpola os dados brutos do IBGE — eles são utilizados diretamente como entrada para os modelos de projeção.

Dados estimados pelo IBGE para anos intercensitários são indicados como tal na interface e podem apresentar variações em relação a revisões futuras realizadas pelo Instituto.

6. Precisão dos Modelos de Projeção

Os modelos de projeção populacional implementados no HidroFlow (Aritmético, Geométrico, Logístico e Holt-Winters) são ferramentas matemáticas de estimativa. Nenhum modelo é capaz de prever com certeza o comportamento demográfico futuro, pois este depende de fatores socioeconômicos, migratórios e políticos imprevisíveis.

O melhor ajuste estatístico (maior R² ou menor RMSE) não garante a melhor projeção para o horizonte de projeto. O profissional responsável deve avaliar o modelo mais adequado para a realidade do município estudado.
