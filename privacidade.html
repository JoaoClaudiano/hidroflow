Política de Privacidade

Última atualização:

Resumo: O HidroFlow não coleta, armazena nem transmite dados pessoais a servidores externos. Todos os cálculos, projetos e configurações são processados e armazenados exclusivamente no seu navegador.

1. Natureza do Aplicativo

O HidroFlow é uma aplicação web client-side (executada inteiramente no navegador do usuário). Não existe back-end próprio, banco de dados remoto ou servidor de processamento. Todo o código-fonte é executado localmente, podendo inclusive ser utilizado offline após o primeiro carregamento.

2. Dados Armazenados Localmente

Os seguintes dados podem ser armazenados no seu navegador via localStorage:

Dado Finalidade Onde é guardado
Projetos salvos Persistência entre sessões ● Navegador (local)
Preferência de tema (claro/escuro) Personalização de interface ● Navegador (local)
Responsável técnico e nome da empresa Inclusão no cabeçalho do relatório ● Navegador (local)

Esses dados ficam apenas no seu dispositivo. Você pode apagá-los a qualquer momento limpando os dados do site no seu navegador.

3. APIs de Terceiros

Para consultar dados censitários e geolocalizar municípios, a aplicação realiza requisições a APIs públicas. Nenhum dado pessoal é enviado — apenas os parâmetros de busca (nome ou código do município):

Serviço Finalidade Dados enviados Tipo
IBGE SIDRA API Dados populacionais (censos 1991–2022) Código de município ⚡ Externo
OpenStreetMap / Nominatim Geocodificação e limites municipais Nome do município e UF ⚡ Externo
Google Fonts Carregamento das fontes IBM Plex Nenhum (requisição de recurso estático) ⚡ Externo

Essas requisições seguem as políticas de privacidade dos respectivos serviços. Recomendamos consultar os termos do IBGE e do Nominatim / OpenStreetMap.

4. Cookies

O HidroFlow não utiliza cookies. As preferências de sessão são armazenadas via localStorage, que não é transmitido automaticamente em requisições HTTP como os cookies.

5. Rastreamento e Analytics

O HidroFlow não incorpora ferramentas de rastreamento, analytics ou publicidade (Google Analytics, Meta Pixel, Hotjar, etc.). Não há coleta de comportamento de navegação, endereços IP ou qualquer identificador de sessão.

6. Segurança dos Dados

Como todos os dados ficam no seu dispositivo, você é responsável pela segurança do ambiente local. Recomendamos não armazenar informações sensíveis nos campos de texto livres (notas, observações) de projetos em dispositivos compartilhados.

7. Menores de Idade

O HidroFlow é uma ferramenta técnica de engenharia direcionada a estudantes e profissionais. Não há coleta intencional de dados de menores de 18 anos.

8. Alterações nesta Política

Esta política pode ser atualizada a qualquer momento. As alterações serão refletidas na data de "Última atualização" no topo desta página. Recomendamos revisitá-la periodicamente.

9. Contato

Dúvidas sobre privacidade podem ser enviadas pela página de contato ou diretamente pelo GitHub Issues.
