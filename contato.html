Perguntas Frequentes

O HidroFlow substitui o EPANET?

Não. O HidroFlow importa e exporta arquivos no formato .INP (compatível com EPANET) e realiza cálculos estáticos pelo método de Hardy-Cross, mas não replica a simulação dinâmica completa do EPANET. A ferramenta é complementar e focada na fase de concepção e pré-dimensionamento.

Os resultados têm validade técnica para projetos executivos?

Os resultados são baseados em métodos normativos reconhecidos (ABNT, FUNASA), mas qualquer projeto executivo deve ser revisado e assinado por engenheiro civil ou sanitarista habilitado no CREA. Veja os Termos de Uso para mais detalhes.

Posso usar o HidroFlow para meu TCC ou dissertação?

Sim! O HidroFlow foi desenvolvido com esse propósito. Ao citar a ferramenta, recomendamos utilizar a referência disponível no Memorial de Cálculo ou o repositório GitHub como fonte.

Meus dados estão seguros?

Sim. O HidroFlow não possui servidor próprio. Todos os projetos ficam no localStorage do seu navegador e nunca são transmitidos a terceiros. Veja a Política de Privacidade para detalhes completos.

Como posso contribuir com o projeto?

Acesse o repositório no GitHub e siga as instruções do arquivo CONTRIBUTING.md. Contribuições de código, documentação, testes e relatos de bugs são bem-vindas.
